# 🚀 MongoDB Department-Employee API
A simple API built with MongoDB to manage:

📂 Departments

👩‍💼 Employees

💰 ETFs

🏗️ Projects

This README guides you through inserting data, creating relationships, and accessing endpoints using Postman.
